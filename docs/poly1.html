実装

高次から低次に向かって配列に係数を格納する。 \[ P(x) = c_1 x^{n-1} + c_2 x^{n-2} + \cdots + c_{n-1} x + c_n \] ホーナー法を使って評価する。 \[ P(x) = (\cdots((c_1 x + c_2) x + c_3) x + \cdots + c_{n-1}) x + c_n \] 微分は \[ P'(x) = c_1 (n-1) x^{n-2} + c_2 (n-2) x^{n-3} + \cdots + c_{n-1} \] 不定積分は \[ \int P(x) dx = \frac{c_1}{n} x^n + \frac{c_2}{n-1} x^{n-1} + \cdots + \frac{c_{n-1}}{2} x^2 + c_n x + C \]
